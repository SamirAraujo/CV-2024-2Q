Projeto de Reconhecimento Facial

Visão Geral

Este projeto cria um sistema que reconhece rostos usando uma webcam. Ele captura vídeo, detecta rostos nos quadros do vídeo e os compara com rostos conhecidos armazenados em uma pasta específica.

Itens Requeridos Para Execução

Para executar o código deste projeto em seu computador, você precisará ter o seguinte:

  • Python 3.x: A versão mais recente do Python instalada em seu sistema. Você pode baixá-la no site oficial do Python.
  • Bibliotecas Necessárias: As seguintes bibliotecas Python devem ser instaladas. Você pode instalá-las usando o pip:
    • face_recognition - Biblioteca para reconhecimento facial.
    • os - Biblioteca padrão do Python para interações com o sistema operacional.
    • sys - Biblioteca padrão do Python para manipulação de parâmetros e funções do sistema.
    • cv2 (OpenCV) - Biblioteca para captura e processamento de vídeo.
    • numpy - Biblioteca para operações matemáticas e manipulação de arrays.
    • math - Biblioteca padrão do Python para funções matemáticas.

Você pode instalar todas as bibliotecas necessárias com o seguinte comando:

				pip install face_recognition opencv-python numpy

Configuração da Pasta

É necessário ter uma pasta criada que contenha imagens dos rostos que você deseja que o sistema reconheça. Cada arquivo de imagem deve ser nomeado com o nome da pessoa na foto.

Componentes Principais

DIRECTORY

Este é o caminho para a pasta com as imagens dos rostos conhecidos.

face_confidence

Esta função calcula o nível de confiança do sistema de que um rosto detectado corresponde a um rosto conhecido.

Parâmetros:

  • face_distance: Quão diferente o rosto detectado é do rosto conhecido.
  • face_match_threshold: O limite para decidir se os rostos correspondem (o padrão é 0.6).

Retorna: Nível de confiança como uma string de porcentagem.

Classe FaceRecognition

Esta é a parte principal do sistema que lida com tudo, desde a codificação dos rostos conhecidos até o reconhecimento dos rostos no vídeo.

__init__

Configura o sistema FaceRecognition e o prepara para reconhecer rostos.

Atributos:

  • known_face_encodings: Lista de codificações de rostos conhecidos.
  • known_face_names: Lista de nomes correspondentes às codificações de rostos conhecidos.
  • face_locations: Lista de localizações de rostos no quadro de vídeo atual.
  • face_names: Lista de nomes dos rostos reconhecidos no quadro de vídeo atual.
  • process_current_frame: Controla o processamento dos quadros.

encode_faces

Codifica os rostos das imagens na pasta especificada e armazena as codificações e nomes.

Atributos:

  • known_face_encodings: Atualizado com codificações de rostos das imagens.
  • known_face_names: Atualizado com nomes extraídos dos nomes dos arquivos de imagem.

run_recognition

Captura vídeo da webcam, detecta rostos e os compara com os rostos conhecidos.

Atributos:

  • video_capture: Captura vídeo da webcam padrão.
  • frame_count: Contador para o número de quadros processados.
  • frame_skip: Número de quadros a serem ignorados entre o processamento (padrão é 15).

Métodos:

  • Lê quadros da webcam.
  • Redimensiona e converte quadros para detecção de rostos.
  • Detecta e codifica rostos nos quadros.
  • Compara os rostos detectados com os rostos conhecidos.
  • Desenha retângulos e etiquetas ao redor dos rostos reconhecidos nos quadros de vídeo.
  • Exibe os quadros de vídeo com anotações.
  • Encerra o loop e libera os recursos quando 'q' é pressionado.

Como Usar

Para iniciar o código, basta executar o script. Isso configurará o objeto FaceRecognition, codificará os rostos da pasta e começará a capturar vídeo e reconhecer rostos.
